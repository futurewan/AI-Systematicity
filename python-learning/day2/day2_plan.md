# 第 2 天计划 - 异常处理 + 列表/字典推导式 + 函数练习

## 今日目标
- 建立 Python 异常处理的信心

2. 列表推导式

3. 字典推导式

- `normalize_scores(scores)` -> 最小-最大归一化到 [0, 1]
- `top_k(nums, k)` -> 返回前 k 个唯一数字
