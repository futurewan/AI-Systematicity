# 第三天学习计划（OOP + 模块化）

## 今日目标
- 掌握 Python 面向对象核心：类、实例、继承、方法重写。
- 学会模块化拆分：把功能放到独立 `.py` 文件并导入使用。
- 建立“可维护代码”的习惯（命名、职责分离、入口函数）。

## 今日必做任务
1. 类与对象
- 创建 `User` 类，包含 `name`、`role` 属性。
- 实现 `introduce()` 方法。

2. 继承与重写
- 创建 `Admin(User)` 子类。
- 重写 `introduce()`，补充管理员身份信息。

3. 模块拆分
- 新建 `utils.py`，写 2 个工具函数：
  - `format_name(name: str) -> str`
  - `is_valid_age(age: int) -> bool`
- 在主文件中导入并调用。

4. 入口规范
- 使用 `if __name__ == "__main__":` 作为脚本入口。

## 验收标准
- 代码能直接运行且无报错。
- 你能解释：
  - 为什么要用继承？
  - 为什么要做模块拆分？
- 每个函数至少有 1 个边界输入测试。
- 完成 `day3_notes.md` 和运行日志 `day3_run.log`。

## 可选加分
- 增加 `__repr__` 或 `__str__`，提升对象打印可读性。
