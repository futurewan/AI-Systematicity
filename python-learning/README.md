# Python 学习冲刺（14 天）

## 目标
- 14 天内建立面向求职的 Python 基础（后端 + AI 应用开发方向）
- 以产出为导向，拒绝被动观看

## 规则
1. 每天必须产出：
   - 可运行的代码
   - 一份简短笔记
   - 一份运行证明（命令输出保存到 logs）
2. 每个主题：学习 1 小时，写代码 2 小时
3. 卡住超过 20 分钟，查文档后继续

## 每日流程
1. 将 `templates/daily-checkin-template.md` 复制到 `dayX/dayX_notes.md`
2. 在 `dayX/` 目录中完成当天任务
3. 运行代码并保存输出到 `logs/dayX_run.log`
4. 更新下方进度表

## 14 天计划
- 第 1-2 天：Python 基础（数据类型、控制流、函数、异常处理）
- 第 3-4 天：面向对象编程、模块、包管理（`venv`、`pip`）
- 第 5-6 天：文件 IO、JSON、日志、正则表达式
- 第 7-8 天：并发基础（线程/进程/asyncio）
- 第 9-10 天：HTTP 基础与 API 调用
- 第 11-12 天：FastAPI 基础（路由、数据校验、错误处理）
- 第 13-14 天：迷你项目（`/chat` API + 日志 + 错误处理）
