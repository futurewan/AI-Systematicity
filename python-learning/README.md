# Python Learning Sprint (14 Days)

## Goal
- In 14 days, build a job-oriented Python foundation for backend + AI application development.
- Focus on deliverables, not passive watching.

## Rules
1. Every day must produce:
- runnable code
- one short note
- one proof of execution (command output in logs)
2. For each topic: learn 1 hour, code 2 hours.
3. If stuck for 20 minutes, search docs and continue.

## Daily Workflow
1. Copy `templates/daily-checkin-template.md` to `dayX/dayX_notes.md`.
2. Complete today's tasks in `dayX/`.
3. Run code and save output to `logs/dayX_run.log`.
4. Update progress table below.

## 14-Day Plan
- Day 1-2: Python basics (data types, control flow, functions, exceptions)
- Day 3-4: OOP, modules, package management (`venv`, `pip`)
- Day 5-6: file IO, JSON, logging, regex
- Day 7-8: concurrency basics (thread/process/asyncio)
- Day 9-10: HTTP basics and API calls
- Day 11-12: FastAPI basics (routes, validation, error handling)
- Day 13-14: mini project (`/chat` API + logging + error handling)
